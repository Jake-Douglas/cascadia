# Pokémon Cascadia

Downloadable Pokémon-style RPG set in the Pacific Northwest, expanding into a reimagined Southern Alberta. Native installers for Windows, macOS, and Linux from a single codebase via Electron. A small browser version exists as a demo only.

All visual assets generated by ChatGPT, all code by Claude.

## Distribution Model

**Cascadia is downloadable software, not a website.** Read `docs/ARCHITECTURE.md` for the reasoning. Quick version: browser saves are too fragile for a game where players invest 100+ hours.

Players download a real native app for their OS:
- Windows: `.exe` installer (proper installer, Start menu shortcut, Add/Remove Programs entry)
- macOS: `.dmg` (drag Cascadia into Applications)
- Linux: `.AppImage` (one portable file, double-click to run)

**No Whiskey, no Wine, no compatibility layer.** Mac users get a real Mac app. Windows users get a real Windows app. This is the same approach Discord, VS Code, Slack, etc. use.

A small browser demo at the eventual cascadia URL lets people try the first 30-60 minutes without installing.

## Run locally (browser, for dev)

```bash
cd cascadia
python3 -m http.server 8000
# open http://localhost:8000
```

## Run locally (desktop, for dev)

```bash
cd cascadia
npm install                    # one-time setup
# In terminal 1:
npm run dev:server
# In terminal 2:
npm run electron-dev
# An Electron window opens running the game with desktop save support
```

## Build distributable installers

```bash
npm run build:win              # Windows installer
npm run build:mac              # Mac installer (requires running on a Mac)
npm run build:linux            # Linux AppImage
npm run build                  # All three
```

Output lands in `dist/`.

## Run battle engine tests (no browser needed)

```bash
node tests/battleTest.mjs
node tests/longBattle.mjs
node tests/abilityTest.mjs
```
